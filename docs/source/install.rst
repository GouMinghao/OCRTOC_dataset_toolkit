Installation
============

.. note::
    
    Only Python 3 is tested.


Dataset
^^^^^^^

Clone the Code
--------------

::

    git clone https://github.com/OCRTOC/OCRTOC_dataset_toolkit.git


Install from Source
-------------------

::

    cd OCRTOC_dataset_toolkit    
    python setup.py install

Download the dataset
--------------------
::

    bash download.sh YOUR_DATASET_ROOT
